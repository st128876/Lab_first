# Lab_first
# Лабораторная работа №1: Обработка результатов физических измерений (вариант б)

Данный репозиторий содержит материалы к лабораторной работе по курсу физического практикума СПбГУ (кафедра физической механики).  
**Тема:** Измерение цифровым омметром сопротивления, задаваемого по стрелочному прибору.  
**Цель работы:** освоение методики многократных прямых измерений и статистической обработки полученных данных.

## 📁 Содержание репозитория

- `main.tex` – основной файл отчёта в формате LaTeX (адаптирован для компиляции в Overleaf).
- `table.txt` – сырые данные 50 измерений сопротивления (в Омах), а также рассчитанные отклонения от среднего и их квадраты.
- `plot2_histogram.gp` – скрипт Gnuplot для построения гистограммы.
- `plot3_spline_freq.gp` – скрипт Gnuplot для построения полигона распределения (сглаженная кривая).
- `plot2_histogram.png`, `plot3_spline_freq.png` – готовые изображения графиков (если не используется pgfplots).
- `Document 1.pdf` – пример оформления таблицы интервалов для гистограммы.

## 🔬 Краткое описание работы

Эксперимент заключался в многократном (50 раз) измерении одного и того же значения сопротивления с помощью цифрового омметра. По полученным данным:

- Вычислено среднее арифметическое значение \( R_{\text{ср}} \).
- Найдены случайные отклонения и их квадраты.
- Оценены дисперсия, среднеквадратичное отклонение (СКО) и погрешность среднего.
- Построена гистограмма распределения результатов (интервалы шириной 0,03 Ом).
- Построен полигон частот (сглаженная кривая распределения).
- Рассчитана суммарная погрешность (приборная + случайная) и представлен окончательный результат.

Все вычисления и графики выполнены в соответствии с рекомендациями к лабораторной работе.
